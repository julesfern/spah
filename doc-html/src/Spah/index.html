    See Default client behaviour to learn what actions Spah performs on initialisation. For more information on Spah, see the main Readme (also available on Github)

    Classes

    • Spah.DOM

    • Spah.SpahQL

      The SpahQL module wraps all operations relating to querying JSON objects, and manipulating data based on their responses. SpahQL wraps a query parser with a matching set of query token types, a query runner and the wrapping objects for query responses (QueryResultSet and QueryResult).

    • Spah.State

      Spah.State is the wrapping class for the state defined by your application. It supports querying and assertions against the state and is also responsible for deep-merging state updates from the server and dispatching modification events to your responders.
