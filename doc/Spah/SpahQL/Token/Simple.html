Overview

An intermediary type extended by all any token with a single value such as a string, number, boolean, or a filter query. All subclasses follow the same rules as Token::Base, plus they must store their basic value in the @value variable.

Direct Known Subclasses

Boolean, ComparisonOperator, FilterQuery, KeyName, Numeric, String
